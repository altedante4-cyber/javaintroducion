Conjuntos (Sets)

Los conjuntos en Python son colecciones no ordenadas de elementos únicos. Se pueden crear usando llaves {} o la función set().

Operaciones de actualización de conjuntos

  • setA.discard(elem): elimina elem si está presente
  • setA.pop(): elimina y devuelve un elemento arbitrario
  • setA.clear(): elimina todos los elementos
  • setA.update(*elements): añade elementos de un iterador
  • setA.difference_update(*elements): elimina elementos encontrados en el iterador
  • setA.intersection_update(*elements): mantiene solo elementos en común
  • setA.symmetric_difference_update(setB): mantiene elementos en setA o setB pero no en ambos

Ejemplo:

>>> setA, setB = {1, 2, 3, 4}, {4, 5, 6}
>>> setA.add(6)
>>> setA
{1, 2, 3, 4, 5}
>>> setA.remove(1)
>>> setA
{2, 3, 4, 5}
>>> setA.discard(1)
>>> setA
{2, 3, 4, 5}

Operaciones de conjuntos

  • setA.union(setB) o setA | setB: unión
  • setA.intersection(setB) o setA & setB: intersección
  • setA.difference(setB) o setA - setB: diferencia
  • setA.symmetric_difference(setB) o setA ^ setB: diferencia simétrica

Ejemplo:

>>> setA, setB = {3.14, 4j, 'hola'}, {'juan', 'hola'}
>>> setA | setB  # Igual que setA.union(setB)
{3.14, 4j, 'juan', 'hola'}
>>> setA & setB  # Igual que setA.intersection(setB)
{'hola'}
>>> setA - setB  # Igual que setA.difference(setB)
{3.14, 4j}
>>> setA ^ setB  # Igual que setA.symmetric_difference(setB)
{3.14, 'juan', 4j}

Cadenas de Caracteres

Las cadenas en Python son secuencias inmutables de caracteres Unicode.

Identificación de cadenas

  • cad.isidentifier(): devuelve True si la cadena es un identificador válido
  • keyword.iskeyword(cad): devuelve True si la cadena es una palabra reservada

Ejemplo:

>>> import keyword
>>> 'nombre_variable'.isidentifier(), keyword.iskeyword('nombre_variable')
(True, False)
>>> 'return'.isidentifier(), keyword.iskeyword('return')
(True, True)

Búsqueda en cadenas

  • cad.find(sub): devuelve el índice de la primera ocurrencia de sub
  • cad.rfind(sub): devuelve el índice de la última ocurrencia de sub
  • cad.index(sub): similar a find pero lanza ValueError si no encuentra
  • cad.rindex(sub): similar a rfind pero lanza ValueError si no encuentra

Ejemplo:

>>> x = 'Para cantar es necesario practicar'
>>> x.index('t')
8
>>> x.rindex('t')
29
>>> x.find('ra')
2
>>> x.find('manta')
-1

Transformación de cadenas

  • cad.lower(): convierte a minúsculas
  • cad.upper(): convierte a mayúsculas
  • cad.title(): convierte a formato título
  • cad.capitalize(): capitaliza la primera letra
  • cad.swapcase(): intercambia mayúsculas y minúsculas
  • cad.casefold(): similar a lower pero más agresivo para comparaciones

Ejemplo:

>>> a = 'ANIMALES SALVAJES'
>>> a.lower()  # Devuelve una cadena en minúsculas
'animales salvajes'
>>> a.upper()  # Devuelve una cadena en mayúsculas
'ANIMALES SALVAJES'
>>> a.title()  # Devuelve una cadena en formato título
'Animales Salvajes'

División y unión de cadenas

  • cad.split(sep): divide la cadena por el separador
  • cad.rsplit(sep): divide desde la derecha
  • cad.splitlines(): divide por líneas
  • cad.partition(sep): divide en tres partes (antes, sep, después)
  • cad.rpartition(sep): partition desde la derecha
  • sep.join(iterable): une elementos del iterable con sep

Ejemplo:

>>> '3,45,21,Juan'.split(',')
['3', '45', '21', 'Juan']
>>> ' -> '.join(map(str, range(5)))
'0 -> 1 -> 2 -> 3 -> 4'

Diccionarios

Los diccionarios son estructuras de datos que almacenan pares clave-valor.

Operaciones básicas

  • d[k] = elem: asigna valor a clave
  • d.update(): actualiza con otro diccionario o iterable
  • del d[k]: elimina clave y valor
  • d.pop(k): elimina y devuelve valor de la clave
  • d.popitem(): elimina y devuelve par clave-valor (LIFO)
  • d.setdefault(k, default): devuelve valor si existe, sino crea con default
  • d.clear(): elimina todos los elementos

Ejemplo:

>>> d = dict(nombre='bicicleta', ruedas=1)
>>> d['ruedas'] = 2  # Actualizando el valor
>>> d
{'nombre': 'bicicleta', 'ruedas': 2}
>>> d2 = {'manillar': 1, 'sillin': 1}
>>> d.update(d2)  # Actualizando con d2
>>> d
{'nombre': 'bicicleta', 'ruedas': 2, 'manillar': 1, 'sillin': 1}

Vistas de diccionarios

En Python 3, los métodos keys(), values() e items() devuelven objetos de vista dinámicos:

Ejemplo:

>>> a = dict(nom='Antonio', edad='134.2')
>>> ks, vs, its = a.keys(), a.values(), a.items()
>>> ks
dict_keys(['nom', 'edad'])
>>> a['nombre'] = a['nom']  # Cambiando la clave
>>> ks  # La vista se actualiza automáticamente
dict_keys(['nom', 'edad', 'nombre'])

Iterables e Iteradores

Los iterables son objetos que se pueden recorrer, mientras que los iteradores son objetos que implementan el protocolo de iteración.

Funciones para trabajar con iterables

  • enumerate(iterable): devuelve pares (índice, elemento)
  • any(iterable): True si algún elemento es verdadero
  • all(iterable): True si todos los elementos son verdaderos
  • map(function, iterable): aplica función a cada elemento
  • filter(function, iterable): filtra elementos según función
  • zip(*iterables): combina múltiples iterables
  • reversed(sequence): devuelve iterador en orden inverso

Ejemplo:

>>> list(enumerate(['a', 'b', 'c']))
[(0, 'a'), (1, 'b'), (2, 'c')]
>>> any([False, 0, '', 1])
True
>>> all([1, True, 'texto'])
True
>>> list(map(lambda x: x**2, [1, 2, 3, 4]))
[1, 4, 9, 16]

Creando un iterador personalizado

Ejemplo:

class MiRange:
    def __init__(self, num_max):
        self.num_max = num_max
        self.num_actual = 0
    
    def __iter__(self):
        return self
    
    def __next__(self):
        if self.num_actual < self.num_max:
            n = self.num_actual
            self.num_actual += 1
            return n
        else:
            raise StopIteration()

>>> list(MiRange(5))
[0, 1, 2, 3, 4]

Listas y Tuplas

Las listas son secuencias mutables, mientras que las tuplas son inmutables.

Operaciones con listas

  • Concatenación: lista1 + lista2
  • Repetición: lista * n
  • Eliminación: del lista[i] o del lista[i:j]

Ejemplo:

>>> a = [1, 2, 3, 4, 5]
>>> a = [True, 2.5] + a + [2, 'texto']  # Concatenación
>>> a
[True, 2.5, 1, 2, 3, 4, 5, 2, 'texto']
>>> x = ['Maria', 'Pepe', 'Juan', 'Ana', 'Paco']
>>> del x[0]  # Eliminar elemento
>>> x
['Pepe', 'Juan', 'Ana', 'Paco']
>>> del x[1:3]  # Eliminar por rebanada
>>> x
['Pepe', 'Paco']

Tuplas

Las tuplas se crean con paréntesis o simplemente separando valores con comas:

Ejemplo:

>>> tuple([1, 2, 3])
(1, 2, 3)
>>> 3, 4, 5  # Tupla implícita
(3, 4, 5)
>>> 45,  # Tupla de un elemento
(45,)

Rangos (Range)

Los rangos son secuencias inmutables de números enteros.

Creación de rangos

  • range(stop): desde 0 hasta stop-1
  • range(start, stop): desde start hasta stop-1
  • range(start, stop, step): desde start hasta stop-1 con paso step

Ejemplo:

>>> list(range(5))
[0, 1, 2, 3, 4]
>>> list(range(2, 8))
[2, 3, 4, 5, 6, 7]
>>> list(range(0, 10, 2))
[0, 2, 4, 6, 8]
>>> list(range(10, 0, -2))
[10, 8, 6, 4, 2]

Atributos y métodos de rangos

  • r.start: valor inicial
  • r.stop: valor de parada
  • r.step: paso
  • r.count(valor): cuenta ocurrencias de valor
  • r.index(valor): devuelve índice de valor

Ejemplo:

>>> x = range(5, 10, 2)
>>> x.start, x.stop, x.step
(5, 10, 2)
>>> x.index(7)
1
>>> x.count(6)
0
>>> x.count(9)
1
