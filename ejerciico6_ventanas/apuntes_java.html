1. Variables dentro vs fuera del Loop

El Problema

// ❌ INCORRECTO: Variable dentro del for
for(int i = 0; i < 5; i++) {
    int contador = 0;    // Se reinicia en CADA iteración
    contador++;
}
System.out.println(contador); // Error: contador no existe aquí


// ✅ CORRECTO: Variable fuera del for
int contador = 0;        // Se mantiene entre iteraciones
for(int i = 0; i < 5; i++) {
    contador++;
}
System.out.println(contador); // Imprime: 5

Pregunta de Reflexión

¿Cuántas veces se imprimirá "Hola" en el código incorrecto?

2. Variables de Estado (boolean)

¿Qué es una variable de estado?

Es una variable que indica EN QUÉ SITUACIÓN estamos. Como un interruptor que cambia el comportamiento del programa.

// Ejemplo: Modo de juego
boolean esTurnoJugador1 = true;

for(Caracter c : movimientos) {
    if (c == 'X') {
        esTurnoJugador1 = !esTurnoJugador1;  // Cambiar turno
    } else {
        if (esTurnoJugador1) {
            // Agregar a puntuación del jugador 1
        } else {
            // Agregar a puntuación del jugador 2
        }
    }
}

Pregunta de Reflexión

Si lees "ABCXDEF", ¿en qué variable se guarda "DEF"?

3. If-Else-If: El Orden Importa

El Orden de las Condiciones

// ❌ PROBLEMA: El else-if nunca se ejecuta
if (c != ',') {
    clave += c;
} else if (c != '\n') {  // ❌ NUNCA llega aquí
    valor += c;
}

// ✅ CORRECTO: Verificar primero los casos especiales
if (c == ',') {
    // Solo cambiar modo
} else if (c == '\n') {
    // Guardar y limpiar
} else {
    // Agregar a clave o valor
}

Regla Importante

Siempre verificar PRIMERO los casos especiales (coma, salto de línea) y después el caso general.

4. HashMap en Java

Concepto Básico

Un HashMap funciona como un diccionario: buscas por CLAVE y obtienes un VALOR.

Concepto Analogía Código
Crear Crear un diccionario vacío HashMap<String,String> dict = new HashMap<>()
Agregar Escribir una palabra dict.put("hola", "hello")
Obtener Buscar una palabra dict.get("hola")
Todas las claves Índice del diccionario dict.keySet()
Todos los valores Todas las definiciones dict.values()

Ejemplo Completo

HashMap<String, String> traductor = new HashMap<>();
traductor.put("hola", "hello");
traductor.put("gato", "cat");
traductor.put("perro", "dog");

String palabra = traductor.get("gato");  // Devuelve "cat"
System.out.println(palabra);             // Imprime: cat

5. Errores Comunes en Java

❌ Mayúsculas/minúsculas: keySet() vs KeySet()
❌ Olvidar inicializar: La variable boolean esValor debe estar FUERA del for
❌ No limpiar después de guardar: Después de put() hay que limpiar las variables
❌ Agregar el delimitador: La coma , no debe ir ni en clave ni en valor

Ejercicio 1: Contar Vocales

Dado el String "Hola Mundo", cuenta cuántas vocales tiene (a, e, i, o, u).

int vocales = 0;

// Escribe tu código aquí

int vocales = 0;

    char c = texto.charAt(i);
    c = Character.toLowerCase(c);
    
    if (c == 'a' || c == 'e' || c == 'i' || c == 'o' || c == 'u') {

    }
}

Ejercicio 4: Traductor Simple con HashMap

Dado "hola,hello,mundo,world", crea un HashMap y busca "hola".

String texto = "hola,hello,mundo,world";
HashMap<String, String> traductor = new HashMap<>();

// Escribe tu código aquí

String busqueda = traductor.get("hola");
System.out.println(busqueda); // hello

String texto = "hola,hello,mundo,world";
HashMap<String, String> traductor = new HashMap<>();

String clave = "";
String valor = "";
boolean esValor = false;

for(int i = 0; i < texto.length(); i++) {
    char c = texto.charAt(i);
    
    if (c == ',') {
        esValor = true;
    } else if (c == ',') {  // Alternativa: verificar segundo ','
        traductor.put(clave, valor);
        clave = "";
        valor = "";
        esValor = false;
    } else {
        if (!esValor) {
            clave += c;
        } else {
            valor += c;
        }
    }
}
// Guardar la última palabra
traductor.put(clave, valor);

String busqueda = traductor.get("hola");
System.out.println(busqueda); // hello
