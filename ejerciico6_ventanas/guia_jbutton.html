📘 Guía Completa de JButton en Java Swing

1. ¿Qué es JButton?

JButton es un componente gráfico de Swing que representa un botón clickeable. Extiende de AbstractButton y hereda de JComponent.

javax.swing.JButton
    └── AbstractButton
        └── JComponent
            └── Container
                └── Component
                    └── Object

Un JButton puede mostrar:

  • Solo texto: new JButton("Guardar")
  • Solo icono: new JButton(icono)
  • Texto e icono: new JButton("Guardar", icono)

2. Crear un Array de JButton

2.1 Array unidimensional

// Declaración e inicialización
JButton[] botones = new JButton[5];

// Llenar cada botón
botones[0] = new JButton("Uno");
botones[1] = new JButton("Dos");
botones[2] = new JButton("Tres");
botones[3] = new JButton("Cuatro");
botones[4] = new JButton("Cinco");

// O con un bucle
String[] nombres = {"Uno", "Dos", "Tres", "Cuatro", "Cinco"};
JButton[] botones = new JButton[nombres.length];

for (int i = 0; i < botones.length; i++) {
    botones[i] = new JButton(nombres[i]);
}

2.2 Array bidimensional (matriz)

// Matriz 3x3 de botones (como una calculadora)
JButton[][] teclado = new JButton[4][4];

for (int fila = 0; fila < 4; fila++) {
    for (int col = 0; col < 4; col++) {
        teclado[fila][col] = new JButton();
        panel.add(teclado[fila][col]);  // Veremos esto después
    }
}

3. Agregar JButton a la Ventana

3.1 Estructura básica de una ventana

import javax.swing.*;

public class MiVentana extends JFrame {
    public MiVentana() {
        setTitle("Mi Ventana");
        setSize(400, 300);
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        
        JPanel panel = new JPanel();  // Contenedor para los componentes
        // Agregar botones al panel...
        setContentPane(panel);         // Vincular panel a la ventana
        
        setVisible(true);
    }
    
    public static void main(String[] args) {
        new MiVentana();
    }
}

4. Layouts (Diseños de Panel)

FlowLayout (por defecto)

Los componentes se acomodan en filas. Si no cabe, pasa a la siguiente línea.

panel.setLayout(new FlowLayout());
// O simplemente: panel = new JPanel();

GridLayout

Componentes en filas y columnas uniformes. Ideal para calculadoras.

panel.setLayout(new GridLayout(4, 3)); // 4 filas, 3 columnas

4.1 FlowLayout

panel.setLayout(new FlowLayout());

// Alineación
new FlowLayout(FlowLayout.LEFT);      // Izquierda
new FlowLayout(FlowLayout.RIGHT);     // Derecha
new FlowLayout(FlowLayout.CENTER);    // Centro

// Con espaciado
new FlowLayout(FlowLayout.LEFT, 10, 20); // hgap=10, vgap=20

4.2 GridLayout

// Calculadora 4x4
panel.setLayout(new GridLayout(4, 4));

// Con espaciado entre botones
panel.setLayout(new GridLayout(4, 4, 5, 5)); // filas, cols, hgap, vgap

4.3 BorderLayout

panel.setLayout(new BorderLayout());

panel.add(botonNorte, BorderLayout.NORTH);
panel.add(botonSur, BorderLayout.SOUTH);
panel.add(botonEste, BorderLayout.EAST);
panel.add(botonOeste, BorderLayout.WEST);
panel.add(botonCentro, BorderLayout.CENTER);

4.4 BoxLayout

panel.setLayout(new BoxLayout(panel, BoxLayout.Y_AXIS));

panel.add(Box.createVerticalGlue());  // Espacio flexible
panel.add(boton1);
panel.add(boton2);

5. ActionListener con Arrays de Botones

5.1 La interfaz ActionListener

public interface ActionListener {
    void actionPerformed(ActionEvent e);
}

5.2 Opción 1: La clase implementa ActionListener

public class MiVentana extends JFrame implements ActionListener {
    private JButton[] botones;
    
    public MiVentana() {
        // ... crear botones ...
        
        // Asignar el listener a cada botón
        for (JButton btn : botones) {
            btn.addActionListener(this);  // "this" es el ActionListener
        }
    }
    
    @Override
    public void actionPerformed(ActionEvent e) {
        // ¿Qué botón se presionó?
        JButton fuente = (JButton) e.getSource();
        
        if (fuente == botones[0]) {
            System.out.println("Botón 1 presionado");
        } else if (fuente == botones[1]) {
            System.out.println("Botón 2 presionado");
        }
    }
}

5.3 Opción 2: ActionListener anónimo

botones[0].addActionListener(new ActionListener() {
    @Override
    public void actionPerformed(ActionEvent e) {
        System.out.println("Botón 1 presionado");
    }
});

5.4 Opción 3: ActionListener con lambda (Java 8+)

botones[0].addActionListener(e -> System.out.println("Botón 1"));
botones[1].addActionListener(e -> System.out.println("Botón 2"));

5.5 Opción 4: Identificar con actionCommand

// Asignar un identificador a cada botón
botones[0].setActionCommand("btn_uno");
botones[1].setActionCommand("btn_dos");

@Override
public void actionPerformed(ActionEvent e) {
    String comando = e.getActionCommand();
    
    switch (comando) {
        case "btn_uno":
            System.out.println("Presionaste botón 1");
            break;
        case "btn_dos":
            System.out.println("Presionaste botón 2");
            break;
    }
}

7. Cómo Identificar el Índice del Botón

7.1 Usando ActionCommand

JButton[] botones = new JButton[5];

for (int i = 0; i < botones.length; i++) {
    botones[i] = new JButton("Botón " + i);
    botones[i].setActionCommand(String.valueOf(i));  // "0", "1", "2"...
    botones[i].addActionListener(this);
}

@Override
public void actionPerformed(ActionEvent e) {
    int indice = Integer.parseInt(e.getActionCommand());
    System.out.println("Botón " + indice + " presionado");
}

7.2 Usando una clase interna

for (int i = 0; i < botones.length; i++) {
    final int indice = i;  // "final" es necesario para usar i dentro del lambda
    botones[i] = new JButton("Botón " + i);
    botones[i].addActionListener(e -> {
        System.out.println("Presionaste el botón " + indice);
    });
}

8.1 MouseListener (clic, entrada, salida)

boton.addMouseListener(new MouseAdapter() {

    }
    
    @Override
    public void mouseEntered(MouseEvent e) {

    }
    
    @Override
    public void mouseExited(MouseEvent e) {
        boton.setBackground(Color.BLUE);  // Cuando el mouse sale
    }

boton.addChangeListener(e -> {
    ButtonModel modelo = e.getSource().getModel();
    if (modelo.isPressed()) {
        System.out.println("Botón presionado");
    }
    if (modelo.isRollover()) {
        System.out.println("Mouse sobre el botón");
    }
});

9. Ejemplo Completo: Calculadora Simple

Calculadora con GridLayout y ActionListener

import javax.swing.*;
import java.awt.*;
import java.awt.event.ActionEvent;
import java.awt.event.ActionListener;

public class Calculadora extends JFrame implements ActionListener {
    private JTextField pantalla;
    private String operador = "";
    private double num1 = 0, num2 = 0;
    private JButton[] botonesNumeros;
    private JButton[] botonesOperadores;
    
    public Calculadora() {
        setTitle("Calculadora");
        setSize(300, 400);
        setDefaultCloseOperation(EXIT_ON_CLOSE);
        setLayout(new BorderLayout());
        
        // Pantalla
        pantalla = new JTextField();
        pantalla.setEditable(false);
        pantalla.setFont(new Font("Arial", Font.BOLD, 24));
        pantalla.setHorizontalAlignment(JTextField.RIGHT);
        add(pantalla, BorderLayout.NORTH);
        
        // Panel de botones
        JPanel panelBotones = new JPanel();
        panelBotones.setLayout(new GridLayout(4, 4, 5, 5));
        
        // Crear botones numéricos
        botonesNumeros = new JButton[10];
        for (int i = 0; i <= 9; i++) {
            botonesNumeros[i] = new JButton(String.valueOf(i));
            botonesNumeros[i].setFont(new Font("Arial", Font.BOLD, 20));
            botonesNumeros[i].addActionListener(this);
            botonesNumeros[i].setActionCommand("num" + i);
            panelBotones.add(botonesNumeros[i]);
        }
        
        // Crear botones de operadores
        botonesOperadores = new JButton[4];
        String[] ops = {"+", "-", "*", "/"};
        for (int i = 0; i < 4; i++) {
            botonesOperadores[i] = new JButton(ops[i]);
            botonesOperadores[i].setFont(new Font("Arial", Font.BOLD, 20));
            botonesOperadores[i].setBackground(Color.LIGHT_GRAY);
            botonesOperadores[i].addActionListener(this);
            botonesOperadores[i].setActionCommand("op" + ops[i]);
            panelBotones.add(botonesOperadores[i]);
        }
        
        // Botón igual y limpiar
        JButton btnIgual = new JButton("=");
        btnIgual.setFont(new Font("Arial", Font.BOLD, 20));
        btnIgual.setBackground(Color.ORANGE);
        btnIgual.addActionListener(this);
        btnIgual.setActionCommand("igual");
        panelBotones.add(btnIgual);
        
        JButton btnLimpiar = new JButton("C");
        btnLimpiar.setFont(new Font("Arial", Font.BOLD, 20));
        btnLimpiar.setBackground(Color.RED);
        btnLimpiar.setForeground(Color.WHITE);
        btnLimpiar.addActionListener(this);
        btnLimpiar.setActionCommand("limpiar");
        panelBotones.add(btnLimpiar);
        
        add(panelBotones, BorderLayout.CENTER);
        setVisible(true);
    }
    
    @Override
    public void actionPerformed(ActionEvent e) {
        String comando = e.getActionCommand();
        
        if (comando.startsWith("num")) {
            String num = comando.substring(3);
            pantalla.setText(pantalla.getText() + num);
        } else if (comando.startsWith("op")) {
            operador = comando.substring(2);
            num1 = Double.parseDouble(pantalla.getText());
            pantalla.setText("");
        } else if (comando.equals("igual")) {
            num2 = Double.parseDouble(pantalla.getText());
            double resultado = 0;
            
            switch (operador) {
                case "+": resultado = num1 + num2; break;
                case "-": resultado = num1 - num2; break;
                case "*": resultado = num1 * num2; break;
                case "/": resultado = num1 / num2; break;
            }
            
            pantalla.setText(String.valueOf(resultado));
        } else if (comando.equals("limpiar")) {
            pantalla.setText("");
            num1 = 0;
            num2 = 0;
            operador = "";
        }
    }
    
    public static void main(String[] args) {
        new Calculadora();
    }
}

10. Tips y Buenas Prácticas

✅ Haz

  • Usa setActionCommand() para identificar botones
  • Usa final cuando uses variables dentro de lambdas
  • Usa GridLayout para botones uniformes
  • Usa FlowLayout para flexibilidad
  • Invoca pack() después de agregar todo

❌ No Haz

  • No mezcles diferentes layouts sin entender cómo funcionan
  • No crees listeners anónimos para cada botón (usa un solo listener)
  • No uses setSize() en componentes individuales, usa setPreferredSize()
  • No olvides invocar setVisible(true) al final
⚠️ Nota importante: Swing no es thread-safe. Si modificas la UI desde otro thread, usa SwingUtilities.invokeLater().
