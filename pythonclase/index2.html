Elementos de Línea

Los elementos de línea como negrita, cursiva, importante, énfasis, superíndice, subíndice, texto marcado, texto eliminado, y HTML se colocan uno al lado del otro dentro de un párrafo.

También podemos tener citas cortas y referencias como elementos de línea.

Ejemplo de código:

<p>
    Texto normal con <strong>énfasis</strong>,
    <em>cursiva</em> y <mark>resaltado</mark>.
</p>

Reglas Importantes

  • Los elementos de bloque pueden contener elementos de línea
  • Los elementos de línea NO pueden contener elementos de bloque
  • Los elementos de bloque ocupan todo el ancho disponible
  • Los elementos de línea ocupan solo el espacio necesario
