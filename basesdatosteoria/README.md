# Quiz sobre Guía de Teoría: Actualización de Bases de Datos

Esta página web contiene un quiz con preguntas de tipo test basadas en el contenido del PDF "Guia_Teoria_UPDATE.pdf".

## Cómo usar

1. Abre el archivo `index.html` en tu navegador web.
2. Responde a las 10 preguntas seleccionando la opción correcta.
3. Haz clic en "Enviar Respuestas" para ver tu puntuación.

## Alternativa: Servidor local

Si prefieres ejecutar un servidor local:

1. Navega al directorio del proyecto.
2. Ejecuta: `python3 -m http.server 8000`
3. Abre tu navegador y ve a `http://localhost:8000`

## Contenido

El quiz cubre temas como:
- Configuración de bases de datos
- Modo seguro en MySQL
- Integridad referencial
- Inserciones avanzadas
- UPDATE con funciones y JOINs
- Manejo de NULLs

## Troubleshooting

- Asegúrate de que tu navegador soporta JavaScript.
- Si el servidor no inicia, verifica que Python esté instalado.
