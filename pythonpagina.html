5. Funci√≥n zfill() - Relleno con Ceros

5.1 ¬øQu√© es zfill()?

La funci√≥n zfill() rellena una cadena con ceros ('0') a la izquierda hasta alcanzar una longitud fija.

5.2 Formateo de Tiempo y Fecha

Horas y Minutos (HH:MM):

hora = 9
minuto = 5
# Resultado: '09:05'
print(f"{str(hora).zfill(2)}:{str(minuto).zfill(2)}")

D√≠as y Meses (DD/MM/AAAA):

dia = 3
mes = 1
# Resultado: '03/01/2025'
print(f"{str(dia).zfill(2)}/{str(mes).zfill(2)}/2025")

5.3 Numeraci√≥n Secuencial

Identificadores con ancho fijo:

# Resultado: '000458'
print(str(id_actual).zfill(6))

Nombres de archivo secuenciales:

for i in range(1, 12):
    # Resultado: 'imagen_01.jpg', 'imagen_02.jpg', ..., 'imagen_11.jpg'
    print(f"imagen_{str(i).zfill(2)}.jpg")
üí° ¬øPor qu√© usar zfill() para archivos?
Para que los archivos se ordenen correctamente:
‚ùå Sin zfill: imagen_1.jpg, imagen_10.jpg, imagen_2.jpg
‚úÖ Con zfill: imagen_01.jpg, imagen_02.jpg, imagen_10.jpg

5.4 Preservaci√≥n de Signos

N√∫meros negativos:

# Resultado: '-00123' (El signo permanece al inicio)

N√∫meros positivos (con signo):

6. Formato de 12 Horas (AM/PM)

6.1 Concepto

El formato de 12 horas divide el d√≠a en dos ciclos de 12 horas cada uno, utilizando los sufijos AM y PM.

üìå Puntos Clave
  • AM (Ante Meridiem): Significa "antes del mediod√≠a". Va desde la medianoche hasta el mediod√≠a.
  • PM (Post Meridiem): Significa "despu√©s del mediod√≠a". Va desde el mediod√≠a hasta la medianoche.
  • El 12: Este es el punto m√°s confuso:
    • 12:00 AM es la Medianoche (el inicio de un nuevo d√≠a)
    • 12:00 PM es el Mediod√≠a (el centro del d√≠a)

6.2 Tabla de Referencia

1:00 AM - 11:59 AM Ma√±ana 01:00 - 11:59
12:00 PM Mediod√≠a 12:00
1:00 PM - 11:59 PM Tarde/Noche 13:00 - 23:59

6.3 Conversi√≥n de 12h a 24h

def convertir_12_a_24(hora_12):
    # hora_12 formato: "03:45PM"
    
    # Extraer componentes
    horas = int(hora_12[:2])
    minutos = hora_12[3:5]
    periodo = hora_12[5:].upper()
    
    # Conversi√≥n
    if periodo == "AM":
        if horas == 12:
            horas = 0  # Medianoche
    else:  # PM
        if horas != 12:
            horas += 12  # Tarde/Noche
    
    return f"{str(horas).zfill(2)}:{minutos}"

# Ejemplos:
print(convertir_12_a_24("03:45PM"))  # 15:45
print(convertir_12_a_24("12:00AM"))  # 00:00
print(convertir_12_a_24("12:00PM"))  # 12:00

6.4 Conversi√≥n de 24h a 12h

def convertir_24_a_12(hora_24):
    # hora_24 formato: "15:45"
    
    horas = int(hora_24[:2])
    minutos = hora_24[3:5]
    
    # Determinar AM/PM
    if horas == 0:
        horas_12 = 12
        periodo = "AM"
    elif horas < 12:
        horas_12 = horas
        periodo = "AM"
    elif horas == 12:
        horas_12 = 12
        periodo = "PM"
    else:
        horas_12 = horas - 12
        periodo = "PM"
    
    return f"{str(horas_12).zfill(2)}:{minutos}{periodo}"

# Ejemplos:
print(convertir_24_a_12("15:45"))  # 03:45PM
print(convertir_24_a_12("00:00"))  # 12:00AM
print(convertir_24_a_12("12:00"))  # 12:00PM
‚ö†Ô∏è Errores Comunes
  • Confundir 12:00 AM con mediod√≠a (es medianoche)
  • Olvidar convertir las 12 PM (no se le suma 12)
  • No considerar que 12 AM son las 00:00 en formato 24h
