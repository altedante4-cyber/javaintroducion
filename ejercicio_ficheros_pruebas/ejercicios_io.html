Ejercicios de Java - java.io.Reader y java.io.Writer

Restricciones: Solo se pueden utilizar las clases base FileReader y FileWriter (que heredan de Reader y Writer respectivamente).
NO se permite: BufferedReader, BufferedWriter, InputStreamReader, OutputStreamWriter ni ninguna otra clase que no herede directamente de Reader o Writer.

Manejo de excepciones: Todos los ejercicios deben manejar apropiadamente las excepciones (IOException, FileNotFoundException, etc.) ya sea con try-catch othrows.

Ejercicio 1: Contador de Líneas

Crear un programa en Java que lea un archivo de texto cuyo nombre se pasará como argumento desde la línea de comandos. El programa debe:

  • Utilizar FileReader para leer el archivo carácter por carácter.
  • Contar el número total de líneas (cada salto de línea \n cuenta como una línea).
  • Mostrar el resultado con el formato: "El archivo [nombre] tiene [X] líneas".
  • Manejar FileNotFoundException si el archivo no existe.
  • Manejar cualquier otra IOException.

Entrada: args[0] = nombre del archivo

Salida: Mensaje con el conteo de líneas

Ejercicio 2: Generador de Registro

Crear un programa que genere un archivo de registro (log) con marca de tiempo. El programa debe:

  • Crear un archivo llamado registro.log utilizando FileWriter.
  • Escribir 10 entradas de registro, cada una con formato: "[YYYY-MM-DD HH:MM:SS] - Mensaje de ejemplo #X"
  • Obtener la fecha y hora usando java.time.LocalDateTime.
  • Agregar un salto de línea después de cada entrada (usar \n o el método newLine() de Writer).
  • Cerrar el escritor correctamente usando try-with-resources.
  • En caso de error, mostrar un mensaje claro.

Archivo de salida: registro.log

Ejercicio 3: Lector de Números a Array

Crear un programa que lea un archivo con números enteros y los almacene en un array. El programa debe:

  • Leer un archivo llamado numeros.txt (un número por línea).
  • Primera pasada: contar cuántos números hay (contar los saltos de línea).
  • Crear un array de enteros del tamaño exacto: int[] numeros = new int[cuenta].
  • Segunda pasada: leer cada línea y convertir a entero con Integer.parseInt().
  • Almacenar cada número en la posición correspondiente del array.
  • Mostrar todos los números separados por comas.
  • Manejar errores de archivo no encontrado y de parsing.

Archivo de entrada: numeros.txt (crear con 15 números)

Ejercicio 4: Copia de Archivo

Crear un programa que copie un archivo a otro usando FileReader y FileWriter. El programa debe:

  • Recibir dos argumentos: archivo origen y archivo destino.
  • Usar FileReader para leer carácter por carácter del origen.
  • Usar FileWriter para escribir cada carácter en el destino.
  • Leer con read() y escribir con write(int c).
  • Preservar todos los caracteres exactamente igual (incluyendo saltos de línea).
  • Contar cuántos caracteres se copiaron.
  • Cerrar ambos recursos correctamente.

Entrada: args[0] = origen, args[1] = destino

Ejercicio 5: Excepción Personalizada

Crear un programa con excepción personalizada para manejo de errores. El programa debe:

  • Crear una clase ArchivoException que extienda de Exception.
  • Crear una clase LectorDatos con método leer(String archivo).
  • El método debe lanzar ArchivoException con mensajes específicos:
    • "El archivo no existe: [nombre]"
    • "El archivo está vacío: [nombre]"
    • "Error de lectura: [detalle]"
  • Verificar si el archivo existe antes de leer.
  • Verificar si el archivo está vacío (0 bytes).
  • En el main, capturar ArchivoException y mostrar mensaje.
  • Probar con archivo inexistente, vacío y válido.

Ejercicio 6: Lector de Nombres a Array

Crear un programa que lea nombres y los almacene en un array de strings. El programa debe:

  • Leer un archivo llamado nombres.txt (un nombre por línea).
  • Primera lectura: contar líneas no vacías (ignorar líneas en blanco).
  • Crear array: String[] nombres = new String[cuenta].
  • Segunda lectura: almacenar cada nombre usando trim().
  • Mostrar el array en orden inverso (índice mayor a menor).
  • Mostrar cuántos nombres se leyeron.
  • Ignorar líneas que solo contengan espacios o estén vacías.

Archivo de entrada: nombres.txt (crear con 10 nombres)

Ejercicio 7: Generador de Reporte

Crear un programa que genere un archivo de reporte formateado. El programa debe:

  • Crear archivo reporte.txt usando FileWriter.
  • Usar arrays para almacenar los datos antes de escribir:
    • String[] productos = {"ProductoA", "ProductoB", ...}
    • int[] cantidades = {10, 25, ...}
    • double[] precios = {15.50, 20.00, ...}
  • Escribir encabezado: "═══════════════════════════" y "REPORTE DE VENTAS"
  • Escribir línea separadora.
  • Usar String.format() para formatear cada línea:
    • Formato: "Producto: X | Cantidad: Y | Precio: $ZZ.ZZ"
  • Escribir un pie con el total.
  • Agregar líneas en blanco para legibilidad.

Archivo de salida: reporte.txt

Ejercicio 8: Analizador de Caracteres

Crear un programa que analice un archivo a nivel de caracteres. El programa debe:

  • Leer un archivo cuyo nombre se pasa como argumento.
  • Usar FileReader con read() para leer carácter por carácter.
  • Contar en un array de 5 enteros:
    • Índice 0: total de letras (a-z, A-Z)
    • Índice 1: total de dígitos (0-9)
    • Índice 2: espacios en blanco
    • Índice 3: saltos de línea (\n)
    • Índice 4: otros caracteres
  • Usar Character.isLetter(), isDigit(), isWhitespace() para clasificar.
  • Detectar fin de archivo cuando read() retorna -1.
  • Mostrar resultados en formato tabular.

Entrada: args[0] = nombre del archivo

Ejercicio 9: Combinador de Archivos

Crear un programa que combine múltiples archivos en uno solo. El programa debe:

  • Recibir argumentos: archivo de salida seguido de N archivos de entrada.
  • Usar FileWriter para el archivo de salida.
  • Para cada archivo de entrada:
    • Crear FileReader y leer carácter por carácter.
    • Escribir línea separadora: "=== Inicio de: [nombre] ==="
    • Copiar todos los caracteres del archivo.
    • Escribir línea separadora: "=== Fin de: [nombre] ==="
    • Agregar línea en blanco entre archivos.
  • Almacenar nombres de archivos en un array de strings.
  • Si un archivo no existe, continuar con los demás y mostrar warning.
  • Mostrar cuántos archivos se combinaron.

Entrada: args[0] = salida, args[1...n] = archivos

Ejercicio 10: Lector de Matriz desde CSV

Crear un programa que lea un archivo CSV y lo almacene en un array bidimensional. El programa debe:

  • Leer archivo matriz.csv donde cada línea es una fila y los valores están separados por comas.
  • Primera lectura: contar filas (líneas no vacías) y columnas (comas por línea + 1).
  • Crear matriz: String[][] matriz = new String[filas][columnas].
  • Segunda lectura: usar split(",") para dividir cada línea en valores.
  • Almacenar cada valor en la posición对应ada de la matriz.
  • Mostrar la matriz en formato tabular:
    • Formato: | valor1 | valor2 | valor3 | ...
  • Calcular y mostrar total de celdas (filas × columnas).
  • Manejar filas con diferente número de columnas.

Archivo de entrada: matriz.csv (crear con formato 3x4)

Notas Adicionales

  • Solo usar FileReader y FileWriter como clases de entrada/salida.
  • Para leer líneas sin BufferedReader, se debe implementar la lógica manualmente:
    • Leer carácter por carácter con read().
    • Acumular en un StringBuilder hasta encontrar '\n'.
    • Procesar la línea cuando se detecta el salto de línea.
  • Usar try-with-resources para cerrar recursos automáticamente.
  • Crear archivos de prueba antes de ejecutar.
